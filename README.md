# Проектная работа №4 Яндекс.Практикума. Место. 

## Описание.

Сайт разработан при помощи HTML и CSS, адаптирован для разных экранов (от 320px и выше). Так же реализован Pop-up с помощью JavaScript, есть возможность редактировать информацию о себе и имя.

## Технологии
1. Flexbox 
2. Grid Layout
3. Относительные величины
4. Медиазапросы
5. JavaScript
